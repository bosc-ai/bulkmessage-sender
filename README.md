# WAchat website

Static multi-page website for a WhatsApp Business Platform technology provider review surface.

## Pages

- `index.html` - public business home page
- `platform.html` - product and permission explanation
- `embedded-signup.html` - customer onboarding page for Meta Embedded Signup
- `app-review-demo.html` - reviewer walkthrough for WhatsApp permissions
- `security.html` - trust and security practices
- `support.html` - support resources
- `contact.html` - business contact page
- `privacy.html` - privacy policy URL
- `data-deletion.html` - data deletion instructions URL
- `terms.html` - terms of service
- `cookies.html` - cookie policy
